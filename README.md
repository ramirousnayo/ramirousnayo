## 🧑‍💻 Sobre mí

Soy consultor independiente con experiencia en **finanzas, ERP y transformación digital**, actualmente finalizando una formación intensiva en **desarrollo de software con Python**. Vengo de un entorno donde el análisis de datos, la gestión de sistemas SAP y la automatización de procesos son el día a día — y ese contexto fue lo que me impulsó a dar el siguiente paso: **aprender a construir las herramientas yo mismo**.

Me especializo en desarrollar **aplicaciones web con Django**, integrando mis conocimientos de negocio con habilidades técnicas para crear soluciones que resuelven problemas reales.

- 🌍 Ubicación: Chile
- 🎯 Buscando: Cargos de Analista Senior, Jefaturas en el area de las finanzas con foco en la automatización.
- 📚 Aprendiendo: Django REST Framework · PostgreSQL · Despliegue en la nube
